# 邦楽歌詞研究 択一トレーナー（第1〜10回）

東京藝大「邦楽歌詞研究」前期（第1回〜第10回）の内容を、**4択の択一問題**で反復練習できる試験対策Web学習アプリです。スマホ利用を想定したモバイルファースト設計で、英語・ドイツ語の学習サイトと同じエンジンを用いています。

## 概要・目的

- 各回の整理版講義ノート／前期試験対策 択一問題集をもとに、**全10回・各25問（計250問）**を収録。
- **正答数の少ない（よく間違える）問題から優先的に出題**し、苦手を効率よくつぶせる。
- 各回の「要点」をクイズ中いつでも参照できる。
- 進捗（最高正答率・苦手リスト）は端末内（localStorage）に保存。

## 機能

- **ホーム**：第1〜10回をカード表示。各カードに主題・タグ・進捗バー。上部に苦手数／習得済み数のサマリー。
- **要点・解説**：各回の要点を表示。クイズ中も「📖 要点を見る」でいつでも確認でき、閉じれば同じ問題に戻れる。
- **択一クイズ**：1問ずつ即時判定。正解はポップ＋紙吹雪、不正解はシェイク。問題・選択肢ともシャッフル。10問ずつのラウンド制で「続きを解く」で次の10問へ。
- **全範囲からランダム**：全10回を混ぜて出題。
- **よく間違える問題を集中学習**：苦手リストを**正答数の少ない順**に出題（連続2回正解で「習得済み」として後回し）。
- **結果画面**：正答率のリング表示と、間違えた問題の復習リスト。

## ローカルで開く

`index.html` をブラウザで開くだけで動作します（ビルド・サーバ不要）。

## 構成

```
邦楽歌詞研究/
├── index.html          エントリ
├── .nojekyll
├── css/style.css       スタイル（学習サイト共通デザイン）
└── js/
    ├── app.js          ビュー管理・画面遷移
    ├── quiz.js          クイズエンジン（ラウンド制・優先度順出題）
    ├── storage.js       進捗・苦手リスト保存（localStorage）
    └── data/
        ├── lectures.js  各回のメタ情報＋要点
        └── questions.js 全250問
```
