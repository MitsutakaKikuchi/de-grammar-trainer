# ドイツ語文法トレーナー（Lektion 1-7）

ドイツ語教科書 Lektion 1〜7 の**格変化・人称変化**を中心に、択一（多肢選択）問題で反復練習できる初学者向けWeb学習アプリです。スマホでの利用を想定したモバイルファースト設計。

## 概要・目的

- 元教材 `ドイツ語テキスト_Lektion1-7_学習資料.md` の **文法解説（Grammatik）** と **練習問題（Übung）** を再構成。
- 読解文は学習対象外として除外。
- Lektion を選び、そのレッスンに沿った択一問題に取り組める。
- ヒントボタン・解説付きで、初学者がつまずかず学べる。
- 進捗（最高正答率）は端末内（localStorage）に保存。

## 機能

- **ホーム**：Lektion 1〜7 をカード表示。各カードに学習テーマと進捗バー。
- **文法解説**：各レッスンの格変化表・要点を表示。
- **択一クイズ**：1問ずつ即時判定。正解はポップ＋紙吹雪、不正解はシェイクで明示。問題・選択肢ともシャッフル。
- **10問ずつ＆続きを解く**：各レッスン全20問を10問ごとに出題。1ラウンド終了後「続きを解く」で次の10問へ。
- **解いている途中でも解説**：クイズ画面の「📖 解説を見る」でいつでも文法解説を確認し、閉じれば同じ問題に戻れる。
- **ヒント＆解説**：各問にヒントボタンと、回答後の詳しい解説＋「覚え方のコツ」を表示。
- **結果画面**：正答率のリング表示（カウントアップ）と、間違えた問題の復習リスト。

## ローカルで開く

`index.html` をブラウザでダブルクリックして開くだけで動作します（ビルド・サーバ不要）。

## GitHub Pages で公開する

1. このフォルダの内容をリポジトリのルートとして GitHub に push する。
2. リポジトリの **Settings → Pages** を開く。
3. **Source** を `Deploy from a branch`、Branch を `main` / `(root)` に設定して保存。
4. 数分後、`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開される。

※ `.nojekyll` を同梱しているため、Jekyll による余計な変換は行われません。

## 構成

```
学習サイト/
├── index.html          エントリ
├── .nojekyll           GitHub Pages の Jekyll 無効化
├── css/
│   └── style.css       スタイル（テーマ・アニメーション）
└── js/
    ├── app.js          ビュー管理・画面遷移
    ├── quiz.js          クイズエンジン
    ├── storage.js       進捗保存（localStorage）
    └── data/
        ├── lektionen.js 各Lektionのメタ＋文法解説
        └── questions.js 全択一問題
```

## 技術

- 依存ライブラリ・ビルド不要の素の HTML / CSS / JavaScript。
- `prefers-color-scheme` でダークモード、`prefers-reduced-motion` でアニメーション抑制に対応。
- データ（解説・問題）とロジック（UI・エンジン）を分離。問題の追加・修正は `js/data/questions.js` を編集するだけ。
